# 📊 Employee Productivity & Cost Leakage Analysis (Excel Dashboard)

## 📌 Project Overview

This project is an interactive Excel dashboard designed to analyze employee productivity, absenteeism impact, and departmental cost leakage.

The objective is to identify high-risk areas, monitor monthly financial leakage trends, and evaluate the relationship between absenteeism and productivity to support better workforce decision-making.

## 🎯 Business Objectives

- Track total cost leakage across departments  
- Identify departments with high financial impact  
- Analyze employee risk categorization (High / Medium)  
- Monitor month-wise leakage trends  
- Understand the relationship between absenteeism and productivity  

## 📈 Key KPIs Tracked

- **Total Cost Leakage:** ₹8,89,457  
- **Average Productivity:** 94%  
- **Total Employees:** 50
  
## 📊 Dashboard Features

- Department-wise Cost Leakage Analysis  
- High-Risk Employee Distribution by Department  
- Cost Leakage by Risk Flag  
- Monthly Cost Leakage Trend Analysis  
- Absenteeism vs Productivity Correlation
  
## 🔍 Key Insights

- HR department contributes the highest cost leakage.  
- High-risk employees account for significantly higher financial impact.  
- Productivity decreases as absenteeism increases.  
- Monthly leakage shows fluctuation with peak observed in June.

## 🛠 Tools & Techniques Used

- Microsoft Excel  
- Pivot Tables  
- KPI Cards  
- Column & Line Charts  
- Scatter Plot (Correlation Analysis)  
- Conditional Formatting
